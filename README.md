# Java Programs 🚀

A collection of Java programs and coding exercises covering core Java concepts, control statements, loops, arrays, strings, object-oriented programming, and data structures. This repository is designed for beginners and students preparing for coding interviews and programming exams.

## 📚 Topics Covered

* Java Basics
* Variables and Data Types
* Operators
* If-Else Statements
* Switch Statements
* While Loop
* For Loop
* Arrays
* Strings
* Methods
* Object-Oriented Programming (OOP)
* Exception Handling
* Collections Framework
* Data Structures and Algorithms

## 🎯 Purpose

This repository helps beginners learn Java programming through practical examples and coding exercises. It can also be used for:

* Java interview preparation
* Coding practice
* Learning Data Structures and Algorithms (DSA)

## 🛠 Technologies Used

* Java
* JDK 8+

## 🚀 Getting Started

1. Clone the repository:

```bash
https://github.com/Kiran460309/Java-Programs
```

```bash

```
